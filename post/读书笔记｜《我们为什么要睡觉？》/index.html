读书笔记｜《我们为什么要睡觉？》

Posted on 2022-10-08，4 min read

此书向我们展示了睡眠的意义，同时强有力地批判了我们压缩睡眠时间的错误做法。

Authors: 马修·沃克
Date finished: 09/07/2022
Fiction?: Non-fiction
Note status: Done
Publisher: 北京联合出版公司
Score: ⭐️⭐️⭐️⭐️
Status: Finished
Type: 科普

🤔Thoughts Before Reading

作为人类每天花费1/3时间的睡眠，是我们必须用心面对的生活问题。现在高压的学习、工作和不规律作息，严重阻碍我们的精神健康，其中一个明显影响因素就是睡眠。就我个人来说，我的睡眠时间随着成长不断递减，且逐渐入睡困难，而且每天起床后没有那种焕然一新的感觉，反而是疲惫。电子媒介、人际关系、学习压力，在我们看来一定程度上影响了我。白天不能提高效率，而到了晚上我心神不宁，感觉睡眠是在浪费时间。我迫切需要找到一个合适的态度面对睡眠这件事，或者说需要全盘改造我的日常作息，并尝试改善睡眠质量。

🚀 The Book in 3 Sentences

  1. 睡眠具有维持机能、保持记忆、康复疾病、激发创造性等不可替代的巨大作用，但政治、经济、文化等各领域都没有给睡眠应有的地位和足够的支持，同时咖啡因、酒精、安眠药、电子荧幕、等恶化了睡眠不足的局面，由此导致现代人从生到死，都没有享受到足时间、高质量的睡眠体验，并付出了高昂的隐藏成本（医疗、保险等），乃至生命的代价（疲劳驾驶等）。

  2. 睡眠百利而无一害，想要发挥睡眠的最大作用，就必须适应睡眠的节奏：睡眠以1.5小时为一个周期，前半夜以非快速眼动时期为主导，后半夜以快速眼动时期为主导，二者缺一不可，故7-8小时的睡眠应该得到保证；补觉是根本上是没有用的，还是得保证每天足时且规律的睡眠。

  3. 除了被动接受未来科技给睡眠检测、条件控制等带来的便利，我们也要主动却改善我们的睡眠环境，政府、企业、个人、社会都应该共同参与到改善睡眠的事业中来，这不仅有利于每一个人的身心健康，也会带了实打实的经济利益。对于个人，睡眠建议如下图所示：

🎨 Impressions

从小就知道睡眠重要，但从未全面、细致地了解，这本书给了一个很好的机会深入了解睡眠，并刺激着手做出改变，努力改善睡眠。

Who Should Read It?

我相信每一个人都应该阅读此书。全社会应该提起对睡眠不足的注意，从我们的经济、文化制度入手，共同努力打造有利于人身心健康的睡眠环境。

☘️ How the Book Changed Me

  • 非常想改变晚睡、晚起的作息。
  • 虽然早睡可能不是非常有可能（工作、学习安排、睡眠环境），但依然想做“晨型人”，最终做到十点睡觉、早点五点起床。但最可能的，依然是23:30-24:00上床睡觉，七点起床。
  • 工作、学习安排是改变的重点，要把重要工作放在上午，晚上十点后停止面对电子荧幕。
  • 集体生活时眼罩、耳塞要安排上。
  • 虽然权衡下依然不想放弃咖啡饮料，但要严格限制咖啡因摄入，下午不再喝咖啡或和低因咖啡为代替。

✍️ My Top 3 Quotes

  • 如果只睡6个小时或更少，你就会缩减通常由睡眠纺锤波为大脑带来的学习能力恢复的福利。
  • 即使是只有20分钟的白天小睡，只要包含足够的非快速眼动睡眠，也可以提供记忆巩固的好处。
  • 教条主义和僵化的思维似乎是妨碍更好的商业实践（比如智慧睡眠）发展的最大障碍。
  • 某些企业领导错误地认为花在工作上的时间等同于任务的完成度和生产力。
